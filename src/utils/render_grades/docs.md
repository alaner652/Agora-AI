# render_grades

將單學期成績渲染成 PNG 圖片。

## 使用方式

```python
from utils.render_grades.index import render

path = render(entries, title="114學年度第2學期", output="output/grades.png")
```

## 參數

| 參數 | 型別 | 預設值 | 說明 |
|------|------|--------|------|
| `entries` | `list[dict]` | — | `get_grades()` 過濾後的單學期成績 |
| `title` | `str` | `"學期成績"` | 標題列文字（建議帶入學期名稱） |
| `output` | `str` | `"output/grades.png"` | 輸出路徑 |
| `scale` | `int` | `2` | 像素倍率（2 = Retina 品質） |

## 版面結構

```
┌──────────────────── 標題列 ─────────────────────┐
│           114學年度第2學期                        │
├─ 科目名稱 ──────┬─ 必選修 ─┬─ 學分 ─┬─ 分數 ─────┤
│  大學英文       │   必修   │   2   │   88       │
│  微積分         │   必修   │   3   │   52       │ ← 紅底（不及格）
│  ...            │          │       │            │
└─────────────────┴──────────┴────────┴────────────┘
```

## 顏色說明

| 狀態 | 背景色 |
|------|--------|
| 及格（奇數列） | 白色 |
| 及格（偶數列） | 淺灰藍 |
| 不及格 | 淺紅 |
