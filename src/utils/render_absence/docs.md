# render_absence

將缺曠資料渲染成 PNG 圖片，僅顯示實際有缺曠的節次欄位。

## 使用方式

```python
from utils.render_absence.index import render

path = render(entries, title="114學年度第2學期 缺曠記錄",
              date_range="1150201 ～ 1150518", output="output/absence.png")
```

## 參數

| 參數 | 型別 | 預設值 | 說明 |
|------|------|--------|------|
| `entries` | `list[dict]` | — | `get_absence()` 的回傳值 |
| `title` | `str` | `"缺曠記錄"` | 標題列文字 |
| `date_range` | `str` | `""` | 副標（日期範圍，空字串則不顯示） |
| `output` | `str` | `"output/absence.png"` | 輸出路徑 |
| `scale` | `int` | `2` | 像素倍率（2 = Retina 品質） |

## 節次欄位

只顯示資料中實際出現的節次，依照固定順序：
`朝會 / 自 / 1–9 / K / A–E`

## 顏色對照

| 假別 | 顏色 |
|------|------|
| 缺曠 | 紅色（salmon） |
| 遲到 | 橘色 |
| 事假 | 藍色 |
| 病假 | 綠色（mint） |
| 公假 | 紫色（lavender） |
| 其他 | 淺灰 |
