# fetch_absence

查詢學生缺曠記錄。與課表（AG222）不同，`ak002_00.jsp` 本身就接受 `fncid` 參數，**不需要**先打 `sys001_00.jsp` gateway。

## 流程

```
Step 1  POST ak_pro/ak002_00.jsp  (fncid=AK002)
        →  取完整表單（含 select#yms + select#leave + 日期選項）

Step 2  POST ak_pro/ak002_01.jsp  (含查詢條件)
        →  取缺曠明細 HTML
```

---

## Step 1：取完整表單

| 方法 | URL |
|------|-----|
| POST | `/tsint/ak_pro/ak002_00.jsp` |

**解析目標**：
- `select#yms` → 學期清單
- `select#leave` → 假別清單

**假別代碼**

| value | label |
|-------|-------|
| `00` | 全部 |
| `1` | 缺曠 |
| `10` | 遲到 |
| `21` | 事假 |
| `22` | 病假 |
| `23` | 公假 |
| `24` | 喪假 |
| `25` | 婚假 |
| `26` | 孕(產)假 |
| `27` | 哺育假 |
| `28` | 防疫假 |
| `29` | 生理假 |
| `31` | 原住民假 |

---

## Step 3：查詢缺曠

| 方法 | URL |
|------|-----|
| POST | `/tsint/ak_pro/ak002_01.jsp` |

**Payload**

| 欄位 | 範例值 | 說明 |
|------|--------|------|
| `yms` | `114,2` | 學期 |
| `leave` | `00` | 假別代碼 |
| `etxt_syear` | `115` | 起始年（民國） |
| `etxt_smonth` | `05` | 起始月（補零） |
| `etxt_sday` | `01` | 起始日（補零） |
| `etxt_eyear` | `115` | 結束年 |
| `etxt_emonth` | `05` | 結束月 |
| `etxt_eday` | `18` | 結束日 |
| `spath` | `""` | 固定空字串 |
| `sdate` | `1150101` | 起始日緊排（年月日） |
| `edate` | `1150518` | 結束日緊排 |

**回傳表格欄位（17 個節次）**

```
朝會 / 自 / 1 / 2 / 3 / 4 / 5 / 6 / 7 / 8 / 9 / K / A / B / C / D / E
```

有值表示該節次有缺曠（值為假別名稱，如「缺曠」、「事假」）；空白（`&nbsp;`）表示正常出席。
